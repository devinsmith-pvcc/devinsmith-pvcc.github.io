    Chapter 6: Programming Languages

Machine Language Instructions ecoded as bit patterns.
Debugging Process of locating and correcting errors.
Assembler Converts mnemonic expressions into machine language.
Assembly Language Mnemonic system for representing programs.
Translator Translates high-level primitives to machine-language programs.
Compiler Translates machine-language to high-level primitives.
Interpreter Works as a translator, but also executes the instructions given.
Cross-Platform Software Software that can operate independantly of device operating system or hardware in order to work on multiple different devices.
Natural Language Languages like English, German, and Latin.
Formal Language Programming languages.
Machine Independence The theory of any machine to run any program, which is currently possible, but with at least some modifactions to programs or machines to ensure compatibility.
High-Level Programming Language Provides more abstractions for the programmer and make it easier for people to read and write a program.
Object-Oriented Programming (OOP) Programming language based around data or object rather than fuctions and logic.
Scripting Language Programming language used to perform administrative tasks rather than develop complex programs.
Data Types:
  1. Integer
  2. Float
  3. String
  1. Numeric data consisting of whole numbers.
  2. Also called Real, is numeric data that may contain values other than whole numbers.
  3. Data represented by non-numeric characters.
Primitive Data Type Data types for primitive programming languages, such as int for integer and char for character.
Data Structure Conceptual shape or arrangement of data.
Python Collection Data Structures
  1. List
  2. Tuple
  3. Set
  4. Dictionary
  1. Used to store multiple items in a single variable. Contained in [].
  2. As a list, however it is ordered and unchangeable. Contained in ().
  3. As a list, however it is unordered, unchangeable, and unindexed. Contained in {}.
  4. Stores data values in key:value pairs. Ordered and changable. Does not allow duplicates. Contained in {}.
Function A set of instructions for performing a task, can be used as an abstract tool by other program units.
Local Variable A variable declared within a function, and can only be referanced by that same fuction.
Global Variable Variable with an unrestricted scope, and can be referanced by any function within the program.
Source Program A program in it's original language.
Object Program A translated program from it's original language to another.
