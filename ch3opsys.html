    Operating Systems Terminology

TERM DEFINITION
Operating System The software the controls the overall operation of the computer including file management, the user interface, and the program execution environment.
Job Queue A list of tasks to be carried out in turn by an operating system
FIFO First-In-First-Out, is an algorithm used in operating systems to manage resources such as CPU time, memory, and input/output operations.
Interactive Processing Allows a program being executed to carry on a dialogue with the user through remote terminals.
Time-Sharing An operating system designed to provide service to multiple users at the same time.
Multiprogramming Dividing time into intervals and then execution of each job is restricted to only one interval at a time.
Application Software Consists of the programs for performing tasks particualr to the machine's utilization.
System Software Performs those tasks that are common to computer systems in general.
Utility Software Consists of software units that extend (or perhaps customize) the capabilites of the operating system.
Graphical User Interface (GUI) A visual way of interacting with a computer using items such as windows, icons, and menus, used by most modern operating systems.
Shell A computer program that exposes an operating system's services to a human user or other programs
Kernel The interal part of an operating system, containing software compontents that perform the very basic functions required by the computer installation.
Device Driver Software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine.
File Manager Software unit used to coordinate the use of the machine's mass storage facilities.
Memory Manager Software unit charged with the task of cooridnating the machine's use of main memory.
Virtual Memory A memort management technique where secondary memory can be used as if it were part of the main memory.
Boot Strapping The procedure that transfers the operating system from mass storage into main memory.
ROM Read-only memory. These memory cells can be read, but cannot be altered.
Deadlock The condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
Spooling A technique used to hold data for output in storage, until a later or more convenient time.
Super User (or Administrator) A user with highly privileged access to the operating system, identified during the login procedure.
Multi-Factor Authentication Additional requirments need to login or gain access, such as a texted code, a fingerprint, or retinal scans.
